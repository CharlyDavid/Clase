ARTICLE

This artice is about Web Page Structure. I have given each elements colours to understand easily how the web page structure works.

All these Elements and Attributes are in fact Containers of other Elements and Attributes and contents we write.

Correct Web Page Structure is very important because an entire web page depends on it or you can say that Web Page Structure is a foundation of Web Page. So, if the web page structure is not correct then the styling of web page will be difficult and out of control, means it will not display as you want it to.
